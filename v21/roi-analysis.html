ROI analysis by assumption set

Current return framing from updated commercial and expert materials.

The ROI case for SolarEX is not one number. It changes with geography, fouling type, electricity value, coating route, and whether the value case is based on monitored uplift or on a framed expectation. This page keeps those assumption sets separated.

These figures should be read as structured commercial models. They are valuable for internal screening, but they depend on the power-price, sunshine-hour, coating-cost, and uplift assumptions attached to each source document.

How to use these figures correctly

  • Use Quartz ROI figures for preliminary screening where dust and wash-burden economics dominate.
  • Use Titan ROI figures only after acknowledging that the value case inherits both the monitored uplift basis and scenario-specific power-price assumptions.
  • Adjust the commercial model when cleaning frequency, labor cost, or water availability are meaningful local cost drivers.
