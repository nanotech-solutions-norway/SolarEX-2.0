# Contributing Standards

## Content governance
All commercial and technical content must follow these rules:
- distinguish between validated study outcomes and general product positioning
- avoid wording that implies guaranteed results in all climates and all systems
- preserve separation between Quartz (SiO₂) and Titan (TiO₂)
- maintain professional industrial tone

## Change categories
- structure
- copy
- technical proof
- styling
- scripts
- SEO

## Pull request checklist
- purpose is clearly defined
- files changed are scoped appropriately
- claims reviewed for accuracy
- navigation impact reviewed
- no broken internal links
- CSS remains consistent with site design language

## Documentation obligations
Update documentation when changing:
- page structure
- primary product positioning
- ROI framing
- proof or evidence sections
