Why this case is commercially useful

The source material does more than show isolated monthly values. It also provides an executive-summary normalisation method: first compare the treated and untreated performance on a per-panel basis, then aggregate the deltas to the array level using actual panel counts. That gives the website a defensible way to discuss portfolio energy uplift without pretending that all arrays behave identically.

Because the treated south array and treated north array behave differently, the case is especially useful for disciplined sales communication. It supports a stronger Quartz proof story while still preserving site-specific nuance.
