Application process

Detailed process view for pilot preparation and field application.

This page expands the implementation diagrams used across the SolarEX website and converts them into a step-by-step operating view.

Application sequence

1. Surface preparation

Thorough cleaning with water and mild detergent, with all oils and residues removed.

2. Safety setup

De-energize panels, ensure ventilation, and use appropriate protective equipment.

3. Coating application

Apply by spray or wipe in a uniform thin layer using the recommended coverage range.

4. Curing process

Allow rapid initial drying and full cure before exposure to rain or aggressive cleaning.
